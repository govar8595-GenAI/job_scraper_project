# 🚀 Python Job Scraper

A Python automation project that extracts job listings and stores them in a CSV file.

---

## 📌 About This Project

This project demonstrates real-world automation using Python.  
It automatically scrapes job listings, saves structured data into CSV format, and captures result screenshots.

---

## ✨ Features

- 🔍 Scrapes job listings automatically
- 💾 Saves results into CSV
- 📸 Captures job result screenshots
- ⚡ Fully automated workflow

---

## 🛠 Tech Stack

- Python
- Playwright / BeautifulSoup
- CSV Handling
- Automation Logic
